* 기간 : 5주 ( 7/13 ~ 8/24 )
* 인원 : 팀 (3인 이하) 프로젝트
* 목적 :
  * 기획부터 개발까지 주체적으로 진행해보는 경험에 목적이 있음
  * **협업**을 경험하는 목적도 있음
    * 트렐로, 잔디, 슬랙, 디코, 깃허브 등등
* 최종 결과 브리핑 : 8/24일 CLC존 회의실
* PPT 포함 내용 :
  * 사용한 프로그램 및 기술
  * 기본 로직 구성 (알고리즘)
  * 프로그램 구동 과정
* **대상 : 기본적으로 언어를 활용해 결과물을 산출하는 프로젝트에 목적이 있으므로, 최소한 한 가지 언어를 모두 배워본 사람을 대상으로 함**
